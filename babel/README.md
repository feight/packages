This package provides a set of dependencies that can be used to run babel with @newsteam/babel-preset without having to do the manual peer dependency install step required if you
install @newsteam/babel-preset directly.

## Usage

### Installation

1. Install the package:

  ```sh
  npm install --save-dev @newsteam/babel
  ```
